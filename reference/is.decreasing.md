# Check if the vector is monotically decreasing

Check if the vector is monotically decreasing

## Usage

``` r
is.decreasing(x)
```

## Arguments

- x:

  ([`numeric()`](https://rdrr.io/r/base/numeric.html)) Vector of numeric
  values

## Value

(`logical(1)`) `TRUE` if the vector is monotonically decreasing, `FALSE`
otherwise
