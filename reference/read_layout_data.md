# Read layout data from a file

Read layout data from a file

## Usage

``` r
read_layout_data(layout_file_path, ...)
```

## Arguments

- layout_file_path:

  Path to the layout file

- ...:

  Additional arguments to pass to the underlying read function

## Value

A matrix with the layout data. The row names are supposed to be letters
A,B,C, etc. The column names are supposed to be numbers 1,2,3, etc.
