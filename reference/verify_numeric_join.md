# Verify Numeric Join

Checks if two numeric values are equal. If either value is `NA`, returns
`TRUE`.

## Usage

``` r
verify_numeric_join(x, y)
```

## Arguments

- x:

  A numeric value to be compared.

- y:

  A numeric value to be compared.

## Value

`TRUE` if the values are equal or if either value is `NA`. Returns
`FALSE` otherwise.
